python print 函数

  2021-12-06   Python   python 字数统计: 819字 阅读时长: 2分

参考文章:

  • https://realpython.com/python-print/
  • https://stackoverflow.com/questions/5082452/string-formatting-vs-format-vs-f-string-literal

print 输出的三种格式比较

1. % 方法

  1. 格式:
    print("hi there %s" % name)
  2. 缺点: 如果 name = (1,2,3),则输出会报错 TypeError,需要改为
    print("hi there %s" % (name,))

2. .format 方法

3. f'{variable name}' 方法

实例

#!/usr/bin/python
sub1 = "python string!"
sub2 = "an arg"

sub_a = "i am a %s" % sub1
sub_b = "i am a {0}".format(sub1)
sub_c = f"i am a {sub1}"

arg_a = "with %(kwarg)s!" % {'kwarg':sub2}
arg_b = "with {kwarg}!".format(kwarg=sub2)
arg_c = f"with {sub2}!"

print(sub_a)    # "i am a python string!"
print(sub_b)    # "i am a python string!"
print(sub_c)    # "i am a python string!"

print(arg_a)    # "with an arg!"
print(arg_b)    # "with an arg!"
print(arg_c)    # "with an arg!"

在 python 解释器输入 print()

  1. 每次输出空白后，会自动换行，因为系统在输出一行的结尾(EOL)会有 newline character.(通常不可见,除了一些文字编辑器)

  2. 查看当前系统的 newline character:

    >>> import os
>>> os.linesep
'\r\n

  3. \r: 把光标移到当前行的开头

  4. 移除 python 的 newline character: rstrip 方法
    'A line of text.\n'.rstrip()

  5. 多行文字 用 ’’’ 包括

    >>> import textwrap
>>> paragraph = '''
...     This is an example
...     of a multi-line string
...     in Python.
...     '''
...
>>> print(paragraph)

    This is an example
    of a multi-line string
    in Python.

# 将上面输出的多行缩进取消
>>> print(textwrap.dedent(paragraph).strip())
This is an example
of a multi-line string
in Python.

  6. 组合不同类型变量

    >>> 'My age is ' + 42
Traceback (most recent call last):

    上述输出报错，因为 python 是强类型语言，不允许将不同类型组合。
    改为: 'My age is ' + str(42)

  7. 输出多项字符串的间隔: sep=' '

  8. 使用 join 来间隔时，注意不同类型不能混用

    >>> print(' '.join(['jdoe is', 42, 'years old']))

    解决方法: 使用星号
    print(*['jdoe is', 42, 'years old'])

打印自定义数据类型

  1. nametuple 标准库
from collections import namedtuple
Person = namedtuple('Person', 'name age')
jdoe = Person('John Doe', 42)
print(jdoe)
>>> Person(name='John Doe', age=42)
  1. 为了将行为添加到 Person 类型 定义一个类
class Person:
    def __init__(self, name, age):
        self.name, self.age = name, age

>>> jdoe = Person('John Doe', 42)
>>> print(jdoe)
<__main__.Person object at 0x7fcac3fed1d0>
  1. 结合使用 namedtuple 和自定义类继承
from collections import namedtuple
class Person(namedtuple('Person', 'name age')):
    pass

__str__和 __repr__

  1. print() 本身使用 str() 于类型转换，在列表和元组中，复合数据类型委托调用 repr()。
class User:
    def __init__(self, login, password):
        self.login = login
        self.password = password
    def __str__(self):
        return self.login
    def __repr__(self):
        return f"User('{self.login}', '{self.password}')"
  1. 如果打印单个对象 User 类，那么您将看不到密码，因为 print(user) 将会通知 str(user)，最终将调用 user.__str__():
>>> user = User('jdoe', 's3cret')
>>> print(user)
jdoe
  1. 如果把 user 放在一个列表里
>>> print(user)
[User('jdoe', 's3cret')]

pprint

from pprint import pprint as pp
>>> print('hello')
hello
>>> pp('hello')
'hello'  # Did you spot the difference?

pprint() 调用 repr() 而不是 str() 用于类型转换

python 打印密码

  1. 要求用户输入密码 input() 这是一个坏主意，因为它会在他们输入时以纯文本形式显示。 在这种情况下，您应该使用 getpass() 函数代替，它屏蔽键入的字符。
from getpass import getpass
password = getpass()
print(password)
