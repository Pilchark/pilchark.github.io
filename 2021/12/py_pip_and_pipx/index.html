python 包管理工具

  2021-12-06   Python   python pip 字数统计: 223字 阅读时长: 1分

系统 pip 用法

设置国内镜像源

  1. 临时设置 : pip install -i https://pypi.tuna.tsinghua.edu.cn/simple [包名]
  2. 永久设置 : pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
  3. 国内镜像网站
    • 阿里云 http://mirrors.aliyun.com/pypi/simple/
    • 中国科技大学 https://pypi.mirrors.ustc.edu.cn/simple/
    • 豆瓣 http://pypi.douban.com/simple/
    • 清华大学 https://pypi.tuna.tsinghua.edu.cn/simple/
    • 中国科学技术大学 http://pypi.mirrors.ustc.edu.cn/simple/

列出已安装的 package

pip list 或 pip freeze

pip检查哪些包需要更新

pip list --outdated

pip升级包

  1. 升级 pip : python -m pip install -U pip
  2. pip install --upgrade 要升级的包名 或
    pip install -U 包名

pipx 用法

  • 参考链接: https://realpython.com/advanced-visual-studio-code-python/#mastering-the-visual-studio-code-user-interface

目的

解决使用 pip 命令安装的包产生的以来冲突问题

安装

  1. pip install -U pipx
  2. pipx ensurepath
  3. pipx 在 ~/.local/pipx/venvs 下安装 pip 包，并在 ~/.local/bin 创建软链接指向 venvs,这样没有安装到全局的包也可以全局使用。
  4. 查看安装的包及其依赖: pipx list
