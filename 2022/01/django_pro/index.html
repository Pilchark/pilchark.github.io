1. hello_project

  • 参考 :
    • 为 docker image 选择合适的 python 源
    • https://cloud.tencent.com/developer/article/1632598

  1. 创建 DockerFile

    # Pull base image
FROM python:3.8-slim-buster
# set environemnt variables
ENV PYTHONDONTWRITEBYTECODE 1
ENV PYTHONUNBUFFERED 1
# Set work directory
WORKDIR /code
# Install dependencies
COPY Pipfile Pipfile.lock /code/
RUN pip install pipenv && pipenv install --system
# Copy project
COPY . /code/

  2. 创建 image : docker build .

  3. 创建 docker-compose.yml

    version: '3.8'
# 注意此 3.8 不表示 python 版本，表示 docker compose 的版本
services:
  web:
    build: .
    command: python /code/manage.py runserver
    volumes:
        - .:/code
    ports:
        - 8000:8000

  4. 启动 container : docker-compose up

  5. 停止服务后避免占用内存 : docker-compose down

  6. 查看服务是否运行 : http://localhost:8000

11. Reviews app

  1. 使用 Foreign Key : 在 Book App 的 models.py 创建 review 类

    class Review(models.Model):
    book = models.ForeignKey(
        Book,
        on_delete=models.CASCADE,
        related_name='reviews',
    )
    review = models.CharField(max_length=255)
    author = models.ForeignKey(
        get_user_model(),
        on_delete=models.CASCADE,
    )
    def __str__(self):
        return self.review

  2. 更新 books/admin.py

    class ReviewInline(admin.TabularInline):
    model = Review

class BookAdmin(admin.ModelAdmin):

admin.site.register(Book, BookAdmin)

def get_queryset(self):

16. 安全性

  1. django 项目部署前的检查 :
    docker-compose exec web python manage.py check --deploy

  2. 创建 docker-compose-prod.yml 从 cocker-compos.yml 拷贝

  3. 使用 特定的 yml 文件运行docker

  4. 更新 settings.py
