python assert 用法

  2022-03-02   Python   python 字数统计: 1065字 阅读时长: 3分
  • 参考文章 :
    • https://realpython.com/python-assert-statement/

assert 的概念

什么是 assert

  1. 除非您的程序中有错误，否则断言条件应始终为真。
  2. 如果条件结果为假，则断言会引发异常并终止程序的执行。

断言的作用

  1. 确保在添加功能和修复代码中的其他错误时不会引入新错误

assert 使用

语法

  1. 一般用法

    number = 42
assert number > 50

  2. 为更明确 assert 的作用，可添加描述性断言

        number = 42
    assert number > 50, f"number > 50 expected, got: {number}"

    在 python 中，assert是一个语句而不是一个函数，其后不能使用括号

  3. 如果你想将一个长断言分成几行，那么你可以使用反斜杠字符 \

        number = 42
    assert number > 0 and isinstance(number, int), \
        f"number greater than 0 expected, got: {number}"

AssertionError 错误

  1. 通常无需显示地指定 AssertionError

一般的 Assert 格式

  1. 比较运算符

  2. assert xx in <list,tuple,set,dict..>

  3. 身份断言 : is / is not

  4. 检查数据类型 : assert isinstance(number, int)

  5. 使用内置的 all() 和 any() 编写用于检查可迭代项中项的真值的断言的函数

    • sd
    assert all([True, True, True])
assert all([True, False, True])
assert any([False, True, False])
assert any([False, False, False])

作为文档工具

  1. 条件不为 true 时, assert 迅速报错并且代码停止运行。

    def get_response(server, ports=(443, 80)):
    assert len(ports) > 0, f"ports expected a non-empty tuple, got {ports}"
    for port in ports:
        if server.connect(port):
            return server.get()
    return None

用 assert 调试的例子

  1. 求圆面积，验证输入的半径

    # circle.py
import math

class Circle:
    def __init__(self, radius):
        if radius < 0:
            raise ValueError("positive radius expected")
        self.radius = radius

    def area(self):
        assert self.radius >= 0, "positive radius expected"
        return math.pi * self.radius ** 2

tire = Circle(42)
print(tire.area())
tire2 = Circle(-42)

注意事项

  1. 使用断言是为了通知开发人员有关程序的错误
  2. 断言不应发出"请求格式"的错误信号，例如 FileNotFoundError

在生产环境中禁用断言以提高性能

  1. 运行 python -O 选项
  2. 设置 PYTHONOPTIMIZE环境变量为适当的值

了解 __debug__ 内置常量

  1. __debug__是一个布尔常量，默认为 True

  2. 它是一个常量， 一旦 Python 解释器运行，就无法更改它的值

  3. __debug__的值取决于 Python 在哪种模式下运行

    • Normal (or debug) 模式(开发环境) : True
    • Optimized(生产环境) : False
    if __debug__:
    if not expression:
        raise AssertionError(assertion_message)
# 等同于
assert expression, assertion_message

  4. 优化模式中会禁用 assert, 所以不能用 assert 来验证数据， 它只是调试和测试中的补充。

    # circle.py
import math

class Circle:
    def __init__(self, radius):
        self.radius = radius

    @property
    def radius(self):
        return self._radius

    @radius.setter
    def radius(self, value):
        if value < 0:
            raise ValueError("positive radius expected")
        self._radius = value

    def area(self):
        return math.pi * self.radius ** 2

    def correct_radius(self, correction_coefficient):
        self.radius *= correction_coefficient

  5. python -O 从生成的编译字节码中删除断言;
    python -OO 比前者多删除文档字符串。

设置 PYTHONOPTIMIZE 环境变量

  1. 将 PYTHONOPTIMIZE 环境变量 设置为非空字符串等效于使用 python -O 命令
    • windows: set PYTHONOPTIMIZE="1"
    • linux : export PYTHONOPTIMIZE="1"

用断言写测试代码

  1. 使用 pytest 测试

使用 assert 的误区

  1. 处理和验证数据

    # store.py
def price_with_discount(product, discount):
    assert 0 < discount < 1, "discount expects a value between 0 and 1"
    new_price = int(product["price"] * (1 - discount))
    return new_price
    • 上述 assert 起到的验证的作用在生产环境中应被 if 或 try 代替。
    def price_with_discount(product, discount):
    if 0 < discount < 1:
        new_price = int(product["price"] * (1 - discount))
        return new_price
    raise ValueError("discount expects a value between 0 and 1")
    • 现在可以把对这个函数的任何调用封装在一个 try … , except捕获的块 ValueError并向用户发送信息性消息

  2. 处理错误

    # Bad practice
def square(x):
    assert x >= 0, "only positive numbers are allowed"
    return x ** 2

try:
    square(-2)
except AssertionError as error:
    print(error)
    • 修改
    # Best practice
def square(x):
    if x < 0:
        raise ValueError("only positive numbers are allowed")
    return x ** 2

try:
    square(-2)
except ValueError as error:
    print(error)

  3. 运行有副作用的操作

    >>> sample = [42, 27, 40, 38]

>>> def popped(sample, index=-1):
...     item = sample.pop(index)
...     return item
...

>>> assert sample[-1] == popped(sample)
>>> assert sample[1] == popped(sample, 1)

>>> sample
[42, 40]
