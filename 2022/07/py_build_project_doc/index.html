使用 MkDocs 构建 python 项目的文档

  2022-07-12   Python   python 字数统计: 888字 阅读时长: 2分
  • 参考文章 : https://realpython.com/dependency-management-python-poetry/

项目概述

  1. 将文档分为 4 个类别的建议

    • Tutorial
    • How-to Guide
    • Reference
    • Explanation

  2. 使用的工具

    • MkDocs : 从 markdown 构建静态页面
    • mkdocstring : 从 docstring 自动生成 markdown
    • Material for MKDocs : 修改 document 样式

先决条件

  1. python 虚拟环境
  2. pip 包管理工具
  3. 完善的 docstring 编写
  4. github/gitee repo

第一步 : 为 构建 Document 创建虚拟环境

mkdir mkdocs-documentation
cd mkdocs-documentation
python3 -m venv venv
source venv/bin/activate
(venv) $ python -m pip install mkdocs
(venv) $ python -m pip install "mkdocstrings[python]"
(venv) $ python -m pip install mkdocs-material

第二步 : 创建一个 python 包项目(或者已有的)

例子 : calculator/
│
├── __init__.py
└── calculations.py

# calculator/calculations.py

def add(a, b):
    return float(a + b)

def subtract(a, b):
    return float(a - b)

def multiply(a, b):
    return float(a * b)

def divide(a, b):
    if b == 0:
        raise ZeroDivisionError("division by zero")
    return float(a / b)

第三步 : 编写和格式化 docstrings

编写 docstring

def greet(name):
    """Print a greeting.

    Args:
        name (str): The name of the person to greet.
    """
    print(f"Hello {name}!")

调用 help(function) 时, python 会打印 docstring.

函数对象的 docstring 保存在 __doc__ 方法中.

扩展 docstring

def add(a, b):
    """Compute and return the sum of two numbers.

    Args:
        a (float): A number representing the first addend in the addition.
        b (float): A number representing the second addend in the addition.

    Returns:
        float: A number representing the artihmetic sum of `a` and `b`.
    """
    return float(a + b)

编写 doctest

def add(a, b):
    """Compute and return the sum of two numbers.

    Examples:
        >>> add(4.0, 2.0)
        6.0
        >>> add(4, 2)
        6.0

    Args:
        a (float): A number representing the first addend in the addition.
        b (float): A number representing the second addend in the addition.

    Returns:
        float: A number representing the artihmetic sum of `a` and `b`.
    """
    return float(a + b)

测试 doctest : python -m doctest calculator/calculations.py

没有输出则说明所有测试通过通过.

使用类型提示

  1. 导入 Union 库, 指定一个参数的多个类型

    from typing import Union

def add(a: Union[float, int], b: Union[float, int]) -> float:
    """Compute and return the sum of two numbers.

    Examples:
        >>> add(4.0, 2.0)
        6.0
        >>> add(4, 2)
        6.0

    Args:
        a (float): A number representing the first addend in the addition.
        b (float): A number representing the second addend in the addition.

    Returns:
        float: A number representing the artihmetic sum of `a` and `b`.
    """
    return float(a + b)

添加模块的 docstrings

docstrings 不限于函数和类, 模块和包也可以使用 docstrings.

模块级: calculator.py

# calculator/calculations.py

"""Provide several sample math calculations.

This module allows the user to make mathematical calculations.

Examples:
    >>> from calculator import calculations
    >>> calculations.add(2, 4)
    6.0
    >>> calculations.multiply(2.0, 4.0)
    8.0
    >>> from calculator.calculations import divide
    >>> divide(4.0, 2)
    2.0

The module contains the following functions:

- `add(a, b)` - Returns the sum of two numbers.
- `subtract(a, b)` - Returns the difference of two numbers.
- `multiply(a, b)` - Returns the product of two numbers.
- `divide(a, b)` - Returns the quotient of two numbers.
"""

包级: __init__.py

# __init__.py

"""Do math with your own functions.

Modules exported by this package:

- `calculations`: Provide several sample math calculations.
"""

第四步 : 使用 MkDocs 准备创建 documentation

创建 MKDocs 项目结构

MkDocs: 允许使用 markdown 创建静态页面

MkDocs 依赖以下结构:

  1. 项目代码
  2. 名为 docs/ 文件夹下的 markdown 文件
  3. 配置文件: mkdocs.yml

在虚拟环境中输入: mkdocs new .

配置 yml 文件

# mkdocs.yml
site_name: Calculation Docs
theme:
  name: "material"

运行 : mkdocs serve

创建 markdown 静态页面

创建 4 个分类

  • docs/tutorials.md
  • docs/how-to-guides.md
  • docs/reference.md
  • docs/explanation.md

编辑页面自定义顺序

# mkdocs.yml
site_name: Calculation Docs

theme:
  name: "material"

nav:
  - index.md
  - tutorials.md
  - how-to-guides.md
  - reference.md
  - explanation.md

自定义页面 title(MKDocs 默认以文件名作为 title)

nav:
  - Calculation Docs: index.md
  - tutorials.md
  - How-To Guides: how-to-guides.md
  - reference.md
  - explanation.md

从 docstrings 输入信息

配置插件信息

# mkdocs.yml
site_name: Calculation Docs

theme:
  name: "material"

plugins:
  - mkdocstrings

nav:
  - Calculation Docs: index.md
  - tutorials.md
  - How-To Guides: how-to-guides.md
  - reference.md
  - explanation.md

Mkdocstrings 允许使用语法糖 ::: 键入 docstrings 信息

在 reference.md 插入 ::: calculator.calculations

只渲染包的 docstring: ::: calculator

第五步: 使用 MkDocs Build

构建并创建 site/ 文件夹

mkdocs build

第六步 : 把 site 部署到 github/gitee
