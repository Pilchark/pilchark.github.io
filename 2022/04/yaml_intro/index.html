了解 yaml

  2022-04-08   Linux   linux 字数统计: 112字 阅读时长: 1分

  • 参考 : https://realpython.com/python-yaml/?utm_source=notification_summary&utm_medium=email&utm_campaign=2022-04-02

  • json 格式的超集

设置转换语法

  1. yaml tag : !!str
    转换后面数据为字符串
  2. !!set : 转换后面数据为集合
  3. !!binary : 二进制
  4. !!python/tuple : 转换为元组

锚和别名

  1. 声明锚点 : &

  2. 饮用锚点 : *

  3. 可使用锚点覆盖属性

    shape: &shape
  color: blue

square: &square
  a: 5

rectangle:
  << : *shape
  << : *square
  b: 3
  color: green

  4. 使用管道符号 “|” 保存多行内容到键
