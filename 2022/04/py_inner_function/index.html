Python 嵌套函数

  2022-04-19   Python   python 字数统计: 1098字 阅读时长: 3分
  • 参考 : https://realpython.com/inner-functions-what-are-they-good-for/

1. 创建 python 嵌套函数

  1. 代码:
def outer_func(who):
    def inner_func():
        print("Hello, {who}!")
    inner_func()

>>> outer_func("World!")

2. 使用嵌套函数的基本方法

2.1. 封装

保护和隐藏一个函数, 无法直接调用内部被隐藏的函数

def increment(number):
    def inner_increment():
        return number + 1
    return inner_increment()

>>> increment(10)
#11
>>> inner_increment()
# error

2.2. 辅助函数功能

# hotspots.py

import csv
from collections import Counter

def process_hotspots(file):
    def most_common_provider(file_obj):
        hotspots = []
        with file_obj as csv_file:
            content = csv.DictReader(csv_file)

            for row in content:
                hotspots.append(row["Provider"])

        counter = Counter(hotspots)
        print(
            f"There are {len(hotspots)} Wi-Fi hotspots in NYC.\n"
            f"{counter.most_common(1)[0][0]} has the most with "
            f"{counter.most_common(1)[0][1]}."
        )

    if isinstance(file, str):
        # Got a string-based filepath
        file_obj = open(file, "r")
        most_common_provider(file_obj)
    else:
        # Got a file object
        most_common_provider(file)

>>> from hotspots import process_hotspots

>>> file_obj = open("./NYC_Wi-Fi_Hotspot_Locations.csv", "r")
>>> process_hotspots(file_obj)
There are 3319 Wi-Fi hotspots in NYC.
LinkNYC - Citybridge has the most with 1868.

>>> process_hotspots("./NYC_Wi-Fi_Hotspot_Locations.csv")
There are 3319 Wi-Fi hotspots in NYC.
LinkNYC - Citybridge has the most with 1868.
  • 此时无论传入文件路径或一个变量都可以正确读取文件

3. 闭包

  1. 定义一个闭包
    • 创建一个嵌套函数
    • 从封闭函数引用变量
    • 返回嵌套函数

3.1. 在闭包中保留状态

  1. 了解闭包工厂函数。闭包是由其他函数返回的动态创建的函数。 他们的主要特点是他们可以完全访问在创建闭包的本地命名空间中定义的变量和名称，即使封闭函数已经返回并完成执行。

  2. 思考下列函数：

    # powers.py
def generate_power(exponent):
    def power(base):
        return base ** exponent
    return power

  3. 实现过程

    • 第 3 行 创建 generate_power()，这是一个闭包工厂函数。 这意味着它每次被调用时都会创建一个新的闭包，然后将其返回给调用者。
    • 第 4 行 定义 power()，这是一个内部函数，接受单个参数， base, 并返回表达式的结果 base ** exponent.
    • 第 6 行 返回 power作为函数对象，而不调用它。

  4. 调用 generate_power() 时：

    • 定义一个新的实例 power()，它接受一个参数 base.
    • 拍摄 power()周围状态的快照， 包括 exponent 以其当前值
    • 返回 power()连同其整个周围的状态
>>> from powers import generate_power
>>> raise_two = generate_power(2)
>>> raise_three = generate_power(3)
>>> raise_two(4)
16
>>> raise_two(5)
25
>>> raise_three(4)
64
>>> raise_three(5)
125

3.2. 自定义闭包状态

  1. 通常，闭包变量对外界完全隐藏。 但是，您可以 提供 getter 和 setter 为它们 内部函数：
def make_point(x, y):
    def point():
        print(f"Point({x}, {y})")
    def get_x():
        return x
    def get_y():
        return y
    def set_x(value):
        nonlocal x
        x = value
    def set_y(value):
        nonlocal y
        y = value
    # Attach getters and setters
    point.get_x = get_x
    point.set_x = set_x
    point.get_y = get_y
    point.set_y = set_y
    return point

4. 使用内部函数添加行为：装饰器

  1. 装饰器描述 decorated_func = decorator(decorated_func)

    @decorator
def decorated_func():
    # Function body...
    pass

  2. 装饰器的妙用

    • 调试 Python 代码的常见做法是插入对 print()检查变量的值，确认代码块被执行，等等。 添加和删​​除调用 print()可能会很烦人，而且您可能会忘记其中一些。 为了防止这种情况，你可以写一个这样的装饰器
    def debug(func):
    def _debug(*args, **kwargs):
        result = func(*args, **kwargs)
        print(
            f"{func.__name__}(args: {args},kwargs: {kwargs}) -> {result}"
        )
        return result
    return _debug

@debug
def add(a, b):
    return a + b

add(5, 6)
# add(args: (5, 6), kwargs: {}) -> 11
# 11

  3. 重新实现 generate_power()作为装饰器函数

    def generate_power(exponent):
    def power(func):
        def inner_power(*args):
            base = func(*args)
            return base ** exponent
        return inner_power
    return power

@generate_power(2)
def raise_two(n):
    return n

raise_two(7)
  • 这个版本的 generate_power()产生与原始实现中相同的结果。 在这种情况下，您使用闭包来记住 exponent和一个装饰器，返回输入函数的修改版本， func().
  • 在这里，装饰器需要接受一个参数 (exponent)，因此您需要有两个嵌套级别的内部函数。 第一级表示为 power()，它将装饰函数作为参数。 第二级表示为 inner_power(), 将args中的参数exponent打包, 对幂进行最终计算，并返回结果。
